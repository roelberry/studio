# **App Name**: Indiana Artivist Directory

## Core Features:

- Artist Gallery: Display a dynamic grid of artists, pulling data from Firestore, with name, image, and tags.
- Artist Profile Pages: Generate individual profile pages dynamically populated with artist details from Firestore.
- Image Gallery: Display multiple examples of the artist's work, to make it easier for other artists to know where to apply to.
- About Page: Show static page that includes project descriptions and artist information
- Links Section: Provide links to each artist's or collective's website, social media, or portfolio.
- Admin Submission Form: Implement a private form for submitting artist data directly to the Firestore 'artists' collection.

## Style Guidelines:

- Primary color: A muted green (#90EE90) to represent growth, harmony, and environmental awareness.
- Background color: A light gray (#F0F0F0), offering a neutral backdrop to allow the artwork to stand out.
- Accent color: A complementary soft brown (#D2B48C) to provide contrast, grounding the design and evoking a sense of history and earthiness.
- Headline Font: 'Belleza' serif to capture an artistic personality.
- Body Font: 'Alegreya' serif to capture an artistic personality.
- Use simple, modern icons to represent links to social media or external sites.
- Employ a responsive grid layout for the Artist Gallery to ensure a seamless experience across devices.
